Déploiement

Mettre en ligne tout le dossier en gardant l'arborescence :
- index.html
- assets/css/style.css
- assets/js/app.js
- assets/img/logo-fly.svg

Pour Vercel : déposer le dossier tel quel. La page d'accueil est index.html.
Pour FileZilla : envoyer l'ensemble des fichiers, pas seulement index.html.
